# Function Plotter for Geometric Brownian Motion

## AMS 320 @ SBU - Quantitative Finance demonstration :D

I'm currently learning an introduction to Stochastic Calculus and wanted to cook up a small simulation. The idea is to create a function plotter with a modular/responsive noise scaling. 

My vision for the final product is that one would input a string of the function they'd like to plot, and then adjust the noise with their mouse/keyboard.

In mathematical notation, that means adjusting the scaling of the stochastic integral: $$\int_0^T f(s) \: \mathrm{d}B$$ by varying $f$.
